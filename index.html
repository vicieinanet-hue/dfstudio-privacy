🔐 Política de Privacidade

DF Studio Pro

Última atualização: 04 de Julho de 2026

O DF Studio Pro ("nós", "nosso" ou "app") respeita sua privacidade. Esta política explica como coletamos, usamos e protegemos suas informações.

📊 Quais dados coletamos?

  • Áudio do microfone: Usamos o microfone para processamento de áudio em tempo real (Auto-Tune, efeitos, gravação). O áudio nunca é enviado para servidores — todo o processamento é feito localmente no seu dispositivo.
  • Arquivos de áudio: Você pode importar MP3, WAV e outros formatos para tocar como playback. Estes arquivos ficam armazenados apenas no seu dispositivo.
  • Configurações do app: Presets, marcadores, mapeamentos de pedal e preferências são salvos localmente no seu dispositivo via SharedPreferences.
  • Nenhum dado pessoal: Não coletamos nome, e-mail, localização, endereço IP ou qualquer dado que possa identificar você.

🔒 Resumo: Todos os dados ficam no seu dispositivo. Zero dados enviados para a nuvem.

🎯 Como usamos seus dados?

  • Processamento de áudio: Para aplicar efeitos de voz, Auto-Tune, equalização e gravação.
  • Playback de música: Para tocar suas músicas importadas como base para performance.
  • Controle remoto (Wi-Fi): O servidor local permite controlar o app de outro dispositivo na mesma rede Wi-Fi — nenhum dado sai da sua rede local.

📱 Permissões

  • RECORD_AUDIO: Necessário para capturar o microfone e processar áudio em tempo real.
  • POST_NOTIFICATIONS (Android 13+): Usado para mostrar notificações de reprodução e controle de áudio.
  • READ_MEDIA_AUDIO: Para importar arquivos de áudio do seu dispositivo (MP3, WAV, etc).
  • INTERNET: Usado exclusivamente para o controle remoto local (mesma rede Wi-Fi) — não há comunicação com servidores externos.
  • BLUETOOTH: Para conectar pedais Bluetooth (opcional) e controlar o app com os pés.

🔒 Armazenamento e Segurança

  • Todos os dados ficam exclusivamente no seu dispositivo.
  • Não usamos servidores externos — zero dados enviados para a nuvem.
  • O áudio processado é descartado após o uso; não armazenamos gravações sem sua ação explícita (botão REC).
  • Os arquivos importados (MP3/WAV) ficam armazenados no diretório do app e podem ser excluídos a qualquer momento.

👶 Dados de Crianças

Nosso app não é direcionado a crianças menores de 13 anos. Não coletamos intencionalmente dados de crianças. Se você é pai/responsável e acredita que seu filho nos forneceu dados, entre em contato para que possamos removê-los.

📝 Alterações nesta Política

Podemos atualizar esta política ocasionalmente para refletir mudanças no app ou em requisitos legais. A data da última atualização estará sempre no topo desta página. Recomendamos revisar periodicamente.
