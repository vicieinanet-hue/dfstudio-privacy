📅 Última atualização: 04 de Julho de 2026

O DF Studio Pro ("nós", "nosso" ou "app") respeita sua privacidade. Esta política explica como coletamos, usamos e protegemos suas informações.

✅ Resumo: Todos os dados ficam exclusivamente no seu dispositivo. Nada é enviado para a nuvem.

📊 Quais dados coletamos?

  • Áudio do microfone: Usamos o microfone para processamento de áudio em tempo real (Auto-Tune, efeitos, gravação). O áudio nunca é enviado para servidores — todo o processamento é feito localmente no seu dispositivo.
  • Arquivos de áudio: Você pode importar MP3, WAV, M4A e outros formatos para tocar como playback. Estes arquivos ficam armazenados apenas no seu dispositivo.
  • Configurações do app: Presets, marcadores, mapeamentos de pedal, preferências de áudio e idioma são salvos localmente via SharedPreferences.
  • Assinaturas (Google Play Billing): Quando você assina um plano, o Google Play gerencia o pagamento. O app recebe apenas o status da assinatura (ativa/inativa) — não recebemos dados de cartão de crédito ou informações de pagamento.
  • Nenhum dado pessoal: Não coletamos nome, e-mail, localização, endereço IP ou qualquer dado que possa identificar você diretamente. (O login Google é opcional e usado apenas para associar sua assinatura).

🔒 Dados sensíveis: O áudio do microfone é processado em tempo real e nunca é armazenado em disco, a menos que você pressione o botão REC para gravar.

🎯 Como usamos seus dados?

  • Processamento de áudio: Para aplicar efeitos de voz, Auto-Tune, equalização, compressão, reverb, delay e gravação.
  • Playback de música: Para tocar suas músicas importadas como base para performance ao vivo.
  • Controle remoto (Wi-Fi local): O servidor local permite controlar o app de outro dispositivo na mesma rede Wi-Fi — nenhum dado sai da sua rede local.
  • Gerenciamento de assinaturas: Para verificar se você tem acesso a recursos Premium (Auto-Tune Pro, presets ilimitados, etc.).
  • Melhorias no app: Usamos dados anônimos de crash (via Firebase Crashlytics) sem identificação pessoal para corrigir bugs.

📱 Permissões solicitadas

🎤 RECORD_AUDIO Capturar microfone para processamento de áudio em tempo real
🔔 POST_NOTIFICATIONS Mostrar notificações de reprodução e controle de áudio (Android 13+)
📂 READ_MEDIA_AUDIO Importar arquivos de áudio do dispositivo (MP3, WAV, etc.)
🌐 INTERNET Exclusivamente para controle remoto local (mesma rede Wi-Fi)
📡 BLUETOOTH Conectar pedais Bluetooth para controle com os pés
💾 MANAGE_EXTERNAL_STORAGE Salvar gravações e backups na pasta pública (opcional, Android 11+)

🔒 Armazenamento e Segurança

  • Todos os dados ficam exclusivamente no seu dispositivo — nada é enviado para servidores externos.
  • O áudio processado é descartado após o uso; não armazenamos gravações sem sua ação explícita (botão REC).
  • Os arquivos importados (MP3/WAV) ficam no diretório do app e podem ser excluídos a qualquer momento.
  • Backups: Se você exportar um backup, ele contém apenas configurações e marcadores — sem áudio.
  • Conexão remota: O WebSocket usa a rede local (mesmo Wi-Fi) e não é acessível da internet.

🛡️ Zero dados na nuvem: Se você desinstalar o app, todos os dados locais são removidos. Não temos acesso a eles.

👶 Dados de Crianças

Nosso app não é direcionado a crianças menores de 13 anos. Não coletamos intencionalmente dados de crianças. Se você é pai/responsável e acredita que seu filho nos forneceu dados, entre em contato para que possamos removê-los.

📝 Alterações nesta Política

Podemos atualizar esta política ocasionalmente para refletir mudanças no app ou em requisitos legais. A data da última atualização estará sempre no topo desta página. Recomendamos revisar periodicamente.
